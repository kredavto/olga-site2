# NOIR & OR — бутик-кофейня

Премиальный одностраничный сайт кофейни: кинематографичный hero, GSAP ScrollTrigger-анимации, плавный скролл Lenis, горизонтальная галерея меню, редакционная вёрстка пекарни, история «от зерна до чашки», masonry-галерея интерьера.

Один самодостаточный файл — `index.html`. Откройте его в браузере или включите GitHub Pages (Settings → Pages → Deploy from branch → main).
